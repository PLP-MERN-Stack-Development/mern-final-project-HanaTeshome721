# EventFlow User Guide (Draft)

## Organizer Onboarding
1. Create an organization, upload branding, invite teammates.
2. Draft events with schedules, ticket tiers, and capacity rules.
3. Publish to activate the EventFlow-hosted landing page or embed widget.

## Attendee Journey
1. Browse events, filter by category/date.
2. Select ticket tier, apply promo codes, and confirm via Stripe checkout.
3. Receive instant confirmation email + QR ticket stored in wallet.

## Day-Of Operations
- Staff launch the kiosk mode (tablet) to scan QR codes and monitor occupancy in real time.
- Waitlisted guests receive push notifications as spots open.

Detailed screenshots and troubleshooting tips will be added as the UI matures.
